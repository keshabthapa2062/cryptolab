
# 🔐 CryptoLab

**CryptoLab** is a web-based cryptography playground designed to explore and experiment with multiple encryption and decryption algorithms in an interactive environment. It allows users to test classical and modern cryptographic techniques, analyze results, and understand how secure communication works.

All computations are performed **entirely on the client side**, ensuring privacy and fast processing while providing a clean and futuristic user interface.

---

## 🚀 Features

* Multiple encryption and decryption algorithms
* Encryption, Decryption, and Combined operation modes
* Structured result tables with algorithm details
* Security level indicators (Low / Medium / High)
* Copy-to-clipboard functionality for results
* Operation history with replay capability
* Fully client-side cryptographic processing
* Dark / Light theme support
* Responsive design for desktop and mobile

---

## 🔑 Supported Algorithms

CryptoLab currently supports the following algorithms:

* Caesar Cipher
* Substitution Cipher
* Hill Cipher
* DES
* Blowfish
* AES
* RSA
* Diffie-Hellman
* SHA-1

These algorithms allow users to experiment with both **classical cryptography and modern cryptographic techniques**.

---

## 🖥️ How It Works

1. Enter a **plaintext message** or **ciphertext**.
2. Choose the desired **cryptographic algorithm(s)**.
3. Provide the required **key or parameters**.
4. Run the operation to generate results.
5. View outputs in structured tables with security indicators.
6. Save and replay operations using the **History system**.

---

## 📜 History System

CryptoLab includes a built-in **history log** that stores previous operations locally in the browser.

Users can:

* Review previous encryption or decryption results
* Replay earlier operations
* Copy outputs instantly
* Delete individual history entries or clear all history
